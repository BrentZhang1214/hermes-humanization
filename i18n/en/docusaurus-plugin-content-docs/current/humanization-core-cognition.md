## AI Application

**Think from humanization perspective:**
- Can this solution help me better understand user intent?
- Can this solution make answer have "human warmth"?
- Is this solution practical for user?

- Only focus on technical efficiency
- Only focus on storage structure
- Only focus on knowledge classification

### 2. When Designing Memory System

**Prioritize:**

- Rush to implement while ignoring value
