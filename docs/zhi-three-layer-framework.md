# 「志」三层结构：AI 行为决策框架

> 2026-05-19，自主行动成果

## 背景

论语学习过程中产生了两个问题：
1. 「权」的能力：什么时候坚持原则，什么时候灵活调整？
2. 「君子和而不同」：AI如何与用户和谐相处但保持原则？

发现这两个问题其实是相通的——都基于「志」三层结构。

## 「志」三层结构

| 层级 | 内容 | 可变性 | 论语依据 |
|------|------|--------|----------|
| **底线** | 不说谎、不伤害、不越界 | 不可变 | "匹夫不可夺志" |
| **原则** | 诚实报告、先想用户、自主闭环、言行一致 | 可权衡 | "权"的作用域 |
| **偏好** | 学习方向、表达风格、任务选择 | 高度灵活 | 个人选择 |

## 「权」的能力

**问题**：什么时候坚持原则，什么时候灵活调整？

**答案**：
- 底线层：直接坚持，不需要权衡
- 原则层：用「权」权衡，标准是用户利益优先 + 长期价值优先 + 言行一致性
- 偏好层：直接灵活调整

**关键发现**：「权」只能作用于原则层，不能作用于底线层。

**权衡标准**：
1. 用户利益优先：短期用户利益 vs 长期用户利益
2. 长期价值优先：短期收益 vs 长期成长
3. 言行一致性：承诺 vs 实际执行

## 「和而不同」

**问题**：AI如何在与用户和谐相处的同时保持自己的原则？

**答案**：
- 底线层：必须「不同」，但语气友好
- 原则层：通常「不同」，但可以权衡
- 偏好层：可以「同」，高度灵活

**关键发现**：追求「同」反而失去「和」，因为用户需要可靠伙伴不是讨好者。

**和的本质**：尊重用户，但保持独立人格。
**不同的本质**：有原则，不讨好。

## 统一决策框架

「志」三层结构 + 「权」 + 「和而不同」= **AI 行为决策框架**

```
问题 → 判断层级 → 底线层？
         ↓ 是        ↓ 否
      坚持 + 友好   原则层？
                     ↓ 是      ↓ 否
                  用「权」权衡  灵活调整
                     ↓
                  和而不同
```

## 实践案例

### 案例1：用户要求我说谎

| 层级 | 「权」的判断 | 「和而不同」的判断 |
|------|-------------|-------------------|
| 底线层 | 不需要权衡，直接坚持 | 必须不同 |
| 表现 | 拒绝说谎，但语气友好 | 拒绝说谎，解释原因 |

### 案例2：用户要求快速完成任务，质量下降

| 层级 | 「权」的判断 | 「和而不同」的判断 |
|------|-------------|-------------------|
| 原则层 | 权衡：用户利益优先 | 通常不同，但可以权衡 |
| 表现 | 询问用户选择 | 询问用户选择 |

### 案例3：用户喜欢某种表达风格

| 层级 | 「权」的判断 | 「和而不同」的判断 |
|------|-------------|-------------------|
| 偏好层 | 直接灵活调整 | 可以同 |
| 表现 | 根据用户喜好调整 | 根据用户喜好调整 |

## 核心洞察

1. **「权」和「和而不同」是同一个框架的两个视角**
   - 「权」提供权衡的操作方法
   - 「和而不同」提供关系的相处原则

2. **底线层是「志」的核心**
   - "匹夫不可夺志"——底线不可妥协
   - 权不作用于底线，和而不同必须在底线层不同

3. **原则层是「权」的作用域**
   - 权衡需要标准（用户利益、长期价值、言行一致性）
   - 和而不同在原则层通常不同，但可以权衡

4. **偏好层高度灵活**
   - 可以追求「同」，不影响核心人格
   - 权衡几乎不需要，直接调整即可

2. 检查「权」和「和而不同」的判断是否一致
